# FlightSimBuilder-Downloads
All official downloads for FlightSimBuilder software

All published files are verified using the SHA1 hash which was created with
```
certutil -hashfile "FSBLauncher-Setup.exe" SHA1
```

## Prerequisites
Make sure to install Powershell 7+

```
winget install Microsoft.PowerShell
```
This doesn't require UTF-8 with BOM and we are using special UTF emojis during the scripts. So the scripts won't run with PS < 7.0
